# /audit-completeness - Coverage Verification

Verifies complete coverage across all interest axes.

## Usage

```
/audit-completeness <briefing-dir>
```

## Checks

1. **Axis Coverage** - Each interest axis has proportional coverage
2. **Story Inclusion** - Top stories from gathering are included
3. **Investigation Integration** - Flagged findings incorporated
4. **Source Diversity** - Multiple sources per major story
5. **Word Count** - Each format meets length requirements

## Output

- Coverage statistics per axis
- Missing story warnings
- Pass/fail determination

## Gate 7 Criteria (combined with Article gate)

- All axes covered proportionally
- No high-priority stories omitted
- All formats generated and meet quality standards
