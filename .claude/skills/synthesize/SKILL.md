# /synthesize - Generate Briefings

Generates three briefing formats from gathered and investigated content.

## Usage

```
/synthesize <briefing-dir>
```

## Output Formats

| Format | Length | Purpose |
|--------|--------|---------|
| short.md | 500-900 words | Executive summary, top stories per axis |
| detailed.md | 2,000-4,000 words | Comprehensive with context and analysis |
| full.md | Unlimited | Complete with sources and methodology |

## Process

1. Load topics, investigations, and verified sources
2. Prioritize stories by significance and relevance
3. Generate each format with appropriate depth
4. Apply neutrality standards throughout

## Neutrality Requirements

- Attribution: "According to [source]..."
- Uncertainty markers: "reportedly", "sources say"
- Multiple perspectives on controversies
- No verdict language (confirmed, debunked, proven)
- No advocacy framing (must, should, clearly)
